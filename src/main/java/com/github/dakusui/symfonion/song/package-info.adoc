A package that hosts classes to model a "song".

A song is structured as follows in **JSON**:

[source, JSON]
----
{
  "$include": [ "includedFile" ],
  "$settings": { "<deviceProfileName>": "..." },
  "$noteMaps": { "<noteMapName>": { "...": "..." }},
  "$parts": { "<partName>": { "...": "..." } },
  "$patterns": { "<patternName>": {"...": "..." }},
  "$grooves": { "<grooveName>": ["..."] },
  "$sequence": { "<partName>": { "...": "..." } }
}
----
